Thème 1
Syndrome Occlusif
⚠️ DÉFINITION CLEF Interruption du transit intestinal → accumulation en amont → urgence chirurgicale.
⚡ Physiopathologie
  • Obstacle → intestin contracte pour pousser le contenu → douleur colique intermittente
  • Chaque contraction consomme de l'énergie → à terme l'intestin s'épuise
  • Accumulation en amont → distension + météorisme (air + liquide + débris)
  • Formation du 3ème secteur → hypovolémie → oligurie → troubles circulatoires
  • Diaphragme refoulé → difficultés respiratoires
  • Germes se multiplient dans la lumière → translocation bactérienne
  • AGM = Arrêt des Gaz et Matières — signe cardinal, PAS critère absolu (partiel possible)
📊 Classification — 2 grands types
🔵 Occlusion MÉCANIQUE
  • Endoluminale : corps étranger, bézoard, parasite, fécalome, polype
  • Pariétale : cancer, volvulus, invagination
  • Extra-pariétale : kyste mésentérique, fibrome utérin, grossesse, pseudo-kyste pancréas
  • BHA augmentés (bruits de lutte)
  • AGM complet
  • Gaz absent dans rectum à l'ASP
🔴 Occlusion FONCTIONNELLE (iléus)
  • Cause vasculaire : thrombose mésentérique → nécrose → plus de péristaltisme
  • Métabolique : hypokaliémie → paralysie intestinale
  • BHA diminués ou silence auscultatoire
  • NHA disséminés, désorganisés, partout
  • Gaz présent dans le rectum
  • Distension globale de tout le tube digestif
STRANGULATION = occlusion mécanique + atteinte vasculaire → ischémie → nécrose → urgence vitale

Repère anatomique : Angle de Treitz = angle duodéno-jéjunal = limite entre occlusion haute et basse.
🧪 Types de bézoards
🌿 Phytobézoard : fibres végétales 💇 Trichobézoard : poils, ongles 💊 Pharmacobézoard : médicaments agglutinés 🥛 Lactobézoard : lait coagulé (nourrisson)

🩺 Sémiologie clinique

SIGNES FONCTIONNELS

  • Douleur colique et intermittente
  • Vomissements (précoces si haute, tardifs fécaloïdes si basse)
  • Distension abdominale
  • AGM (arrêt des gaz et matières)
  • Météorisme

EXAMEN PHYSIQUE

  • Inspection : distension symétrique (grêle) ou asymétrique (volvulus), ondes péristaltiques visibles
  • Palpation : pas de contracture, masse possible. 95% du diagnostic
  • Percussion : hypertympanisme
  • Auscultation : BHA de lutte ↑ (mécanique) / silence (fonctionnelle / strangulation)
  • TR : ampoule vide (méca) / gaz présent (fonctio)
NB important : L'examen physique donne 95% du diagnostic. L'occlusion n'est PAS synonyme de contracture.

QCM — Syndrome Occlusif (10 questions)
Testez vos connaissances sur ce thème
Question 1 / 10
Quelle est la principale caractéristique de la douleur dans le syndrome occlusif ?
✅ Correct ! L'intestin se contracte de façon répétée contre l'obstacle → douleur par intermittence (colique). Entre les contractions, la douleur disparaît.
❌ La bonne réponse est B. La douleur est colique car l'intestin contracte répétitivement pour essayer de franchir l'obstacle.
Question 2 / 10
La formation du 3ème secteur dans l'occlusion intestinale entraîne :
✅ Correct ! Le 3ème secteur est un espace liquidien formé dans la lumière intestinale distale, soustrayant du volume à la circulation → hypovolémie → oligurie → choc.
❌ C est correct. Le liquide séquestré dans les anses distendues ne circule plus → hypovolémie réelle.
Question 3 / 10
Quelle est la 1ère cause d'occlusion intestinale dans le monde ?
✅ Correct ! Les adhérences post-opératoires (brides) sont la 1ère cause mondiale d'occlusion intestinale (occlusions sur brides).
❌ C est correct. Les brides/adhérences post-op sont la 1ère cause mondiale d'occlusion.
Question 4 / 10
Sur l'ASP debout de face, quelle image signe une occlusion MÉCANIQUE ?
✅ Correct ! Mécanique = NHA organisés en pile de monnaie (localisés) + absence de gaz dans le rectum (l'obstacle empêche le transit).
❌ B est correct. Occlusion mécanique → NHA organisés + pas de gaz dans rectum. Fonctionnelle → NHA désorganisés partout + gaz dans rectum.
Question 5 / 10
Concernant l'occlusion haute (au-dessus de l'angle de Treitz), quelle affirmation est vraie ?
✅ Correct ! Occlusion haute : le contenu est rapidement évacué par vomissements (précoces, nombreux). L'intestin en amont est court → distension minimale.
❌ B est correct. Haute = Vx précoces +++ / distension minimale. Basse = Vx tardifs fécaloïdes / distension +++.
Question 6 / 10
La strangulation intestinale se définit comme :
✅ Correct ! Strangulation = occlusion mécanique COMPLIQUÉE d'atteinte vasculaire. Urgence chirurgicale absolue. Si nécrose → plus de péristaltisme → silence auscultatoire.
❌ C est correct. Strangulation = obstruction + compression du pédicule vasculaire → ischémie → nécrose si non traité.
Question 7 / 10
Quel est le meilleur examen en 1ère intention pour une occlusion intestinale (hors invagination) ?
✅ Correct ! L'ASP debout de face est l'examen de 1ère intention pour l'occlusion intestinale. Pour l'invagination : échographie en 1er choix.
❌ A est correct. ASP = examen de choix en occlusion. Exception : invagination → échographie en 1ère intention.
Question 8 / 10
Lors d'une occlusion fonctionnelle, comment se présente l'auscultation abdominale ?
✅ Correct ! Occlusion fonctionnelle = l'intestin ne se contracte plus (paralysie) → silence auscultatoire. À différencier de la mécanique avancée (strangulation) qui peut aussi se compliquer de silence.
❌ B est correct. Fonctionnelle = silence auscultatoire (pas de péristaltisme). Mécanique (début) = bruits de lutte augmentés.
Question 9 / 10
Parmi les bézoards, lequel est constitué de fibres végétales ?
✅ Correct ! Phyto = végétal (fibres végétales). Tricho = poils/cheveux/ongles. Pharmaco = médicaments. Lacto = lait coagulé (nourrisson).
❌ B est correct. Phyto = végétal. Mémorisez : Phyto/végétal — Tricho/poils — Pharmaco/médicaments — Lacto/lait.
Question 10 / 10
L'alcalose métabolique est une complication retrouvée dans :
✅ Correct ! Occlusion haute → vomissements de liquide gastrique acide (HCl) → alcalose métabolique. Occlusion basse → perte de liquide alcalin → acidose métabolique.
❌ A est correct. Haute → perte d'HCl par Vx → alcalose métabolique. Basse → perte de bicarbonates → acidose.

Thème 2
Invagination Intestinale
⚠️ DÉFINITION Une anse intestinale s'invagine dans une anse adjacente → occlusion + ischémie. Urgence diagnostique et thérapeutique.
🧒 Chez le nourrisson (<2 ans)
  • Idiopathique dans 95% des cas
  • Plaques de Peyer hypertrophiées
  • Après infection virale : Rotavirus Rougeole Oreillons
  • Moins de 2 ans : pas de cause retrouvée
👤 Chez l'adulte (>2 ans)
  • Secondaire dans 95% des cas → toujours chercher une cause
  • Jeune adulte : diverticule de Meckel, polype, duplication intestinale
  • >50 ans : cancer, lymphome (chercher systématiquement)
  • Vaccin ROR peut favoriser chez jeune enfant

Selles en gelée de groseille = mélange sang + mucus = signe PATHOGNOMONIQUE de l'invagination chez le nourrisson
Échographie : image en boudin (sandwitch) — signe d'invagination
Lavement baryté : image en cupule / bec d'oiseau — diagnostic et thérapeutique
🔬 Diagnostic
  • Échographie → image en boudin / sandwitch + déshabituation de la FID = examen de choix
  • ASP → si pas d'écho disponible : déshabituation FID, distension des anses
  • Lavement baryté ou hydrosoluble → image en cupule/bec d'oiseau ; à la fois DIAGNOSTIQUE et THÉRAPEUTIQUE
  • TDM et échographie : aussi utiles pour bilan d'extension chez l'adulte
Diagnostic différentiel : Gastro-entérite (PAS d'AGM ni de selles sanglantes), volvulus, appendicite, hernie étranglée, méningite-encéphalite, coliques

Question 8/10

❌ B est correct. Non traitement → ischémie → nécrose → perforation → péritonite.

Thème 3
Volvulus
⚠️ DÉFINITION Torsion d'une anse intestinale ou du côlon sur son axe mésentérique → occlusion mécanique + compromission vasculaire.
📍 Anatomie clef
  • Mésentère : péritoine reliant les anses grêles à la paroi postérieure
  • Mésocôlon sigmoïde : relie le côlon sigmoïde à la paroi post
  • Mésocôlon transverse : suspend le côlon transverse à la paroi post
  • L'anse se tord SUR son axe mésentérique ou méso-colonique
👴 Épidémiologie
  • Surtout chez les vieillards
  • Beaucoup plus fréquent chez les hommes
  • Facteurs favorisants : constipation chronique, mégacôlon, alimentation riche en fibres
  • Volvulus du sigmoïde = le plus fréquent
⚡ Conséquences physiopathologiques
  • Torsion → accumulation air, liquide, débris alimentaires en amont
  • Distension progressive → anse complètement collabée si strangulation
  • Compromission vasculaire à l'endroit de la torsion → ischémie → nécrose → péritonite
  • Signes de gravité : contracture, résistance, rebond = signe d'atteinte péritonéale
🩺 Sémiologie clinique
  • Douleur abdominale violente et brutale
  • AGM complet
  • Distension asymétrique ★ (signe évocateur du volvulus)
  • Vomissements moins précoces que dans le grêle
  • Percussion : hypertympanisme
  • Auscultation : bruits de lutte → silence auscultatoire
  • Signes péritonéaux si nécrose (contracture, rebond)
  • État général rapidement altéré
🔬 Paraclinique
  • ASP : NHA + image spécifique du volvulus (grande anse distale en U inversé)
  • Lavement baryté : image en bec d'oiseau / spirale = signe du volvulus
  • TDM : confirme le diagnostic, bilan d'ischémie
  • Biologie : leucocytose, ionogramme, déshydratation
Image en bec d'oiseau au lavement baryté = signe du volvulus du sigmoïde
ASP : Volvulus du sigmoïde — grande anse distale, image en grain de café
ASP : image en grain de café — volvulus du sigmoïde

Les brides/adhérences représentent :
✅ Les brides/adhérences sont la 1ère cause d'occlusion dans le monde (occlusion sur brides = post-opératoire par fibrose).

Question 5/10
Le mécanisme de formation des brides est :
✅ Toute chirurgie → cicatrisation → fibrose → l'anse peut adhérer à la cicatrice et créer une bride qui comprime → occlusion.
❌ B est correct. Brides = cicatrisation par fibrose post-chirurgicale.

Les signes de gravité dans le volvulus incluent TOUS sauf :
✅ La distension asymétrique seule est un signe évocateur du volvulus mais PAS de gravité. Les signes de gravité sont ceux de nécrose/péritonite : contracture, rebond, fièvre, choc.

Question 7/10
Quel est l'examen paraclinique à demander en 1ère intention devant un volvulus ?
✅ ASP en 1ère intention. Pour le volvulus : l'ASP couché de face peut montrer la grande anse distale en U. Le lavement baryté confirme et peut tenter une réduction.
❌ A est correct. ASP = toujours 1ère intention dans l'occlusion (sauf invagination).
Question 8/10
Quelle structure mésentérique est impliquée dans le volvulus du grêle ?
✅ Mésentère = pour les anses grêles. Mésocôlon sigmoïde = pour le côlon sigmoïde. Mésocôlon transverse = pour le côlon transverse.
❌ A est correct. Grêle = mésentère. Sigmoïde = mésocôlon sigmoïde. Transverse = mésocôlon transverse.
Question 9/10
Dans le volvulus, le silence auscultatoire survient quand :
✅ Au début : bruits de lutte (l'intestin contracte contre l'obstacle). Après nécrose ou strangulation avancée : silence auscultatoire (l'intestin ne contracte plus).
❌ B est correct. Début = bruits de lutte. Tardif/nécrose = silence auscultatoire.
Question 10/10
En cas de signe péritonéal dans un volvulus, cela signifie :
✅ Nécrose → perforation → contenu septique dans le péritoine → irritation péritonéale → contracture/rebond. C'est une urgence chirurgicale absolue.
❌ B est correct. Signe péritonéal = nécrose/perforation → urgence chirurgicale vitale.

Thème 4
Brides & Adhérences
📌 Définition
  • 1ère cause d'occlusion intestinale dans le monde
  • Provoquées par cicatrisation post-opératoire (fibrose)
  • L'anse adhère à la cicatrice → compression → occlusion
  • La cicatrisation se fait toujours par fibrose
⚠️ Mécanisme
  • Toute chirurgie abdominale = risque de brides
  • Inflammation post-op → fibrose → adhérence anse/paroi
  • L'anse liée à la cicatrice → comprimée → lumière rétrécie ou fermée
  • Tableau d'occlusion mécanique (classique)
  • Antécédent chirurgical = indice clef !
Devant toute occlusion intestinale, toujours interroger sur les antécédents chirurgicaux → cicatrice abdominale visible = penser aux brides en 1er

Q2. Évaluer l'étendue avec la règle des 9.
→ Thorax antérieur = 9% + bras droit = 9% + bras gauche = 9% = SCB = 27%. Brûlure grave (>20% = grave chez l'adulte).
Q3. Quel agent a causé ces brûlures et quel mécanisme ?
→ Soude caustique = BASE → mécanisme de saponification (transformation des graisses en savon) + pénétration profonde des tissus. Les brûlures alcalines sont plus profondes que les brûlures acides.
Cas 7 Syndrome tumoral de la fosse iliaque droite
Mr Yassine, 70 ans, fumeur, sans ATCD notable. Consulte pour sang dans les selles depuis quelques mois. Difficultés à la défécation, selles plus fines. Légère asthénie. À l'examen : masse palpable en FID, légèrement douloureuse, d'aspect solide, non pulsatile.
Q1. Quel diagnostic probable ?
→ Cancer du côlon ascendant (caecum/côlon droit) probable. Homme âgé + sang dans selles + selles fines + masse FID = cancer du côlon jusqu'à preuve du contraire.
Q2. Examen paraclinique de confirmation ?
→ Coloscopie totale avec biopsies = gold standard pour le cancer du côlon. Permet biopsie + diagnostic histologique. Compléter par TDM thoraco-abdomino-pelvien pour bilan d'extension.
Q3. Quels autres organes peuvent donner une masse en FID ?
→ Appendice (plastron appendiculaire), caecum, tératome, kyste de l'ovaire droit, abcès. La FID contient : appendice + caecum + partie terminale de l'iléon + ovaire droit (femme).
🎯 Pronostics d'examen & Conseils stratégiques

Basé sur l'analyse du cours : ce que le Dr Lubin a mis en évidence, les cas examinés antérieurement, et la structure des 3 documents.

1
⭐⭐⭐ SYNDROME OCCLUSIF — La base de tout
★★★★★
C'est le CŒUR du cours (24 pages sur Chirurgie 1 + révisions en Session 2). Le prof consacre plus de temps à ce thème. Attendez-vous à un QCM ou cas clinique sur la distinction mécanique vs fonctionnelle, et la lecture de l'ASP (NHA organisés vs désorganisés, gaz dans le rectum). Mémorisez : examen physique = 95% du diagnostic.
2
⭐⭐⭐ INVAGINATION — Cas clinique nourrisson très probable
★★★★★
Le prof a donné beaucoup d'exemples cliniques sur l'invagination avec le tableau nourrisson. Attendez-vous à un cas clinique du type : "nourrisson, pleurs paroxystiques, selles gelée de groseille, que faites-vous ?". Réponse type : échographie → lavement thérapeutique. La différence idiopathique (<2 ans) vs secondaire (>2 ans) est un QCM facile.
3
⭐⭐⭐ HERNIES ÉTRANGLÉES — Urgence chirurgicale
★★★★☆
Le prof a insisté sur la distinction hernie réductible / incarcérée / étranglée. Les cas cliniques d'hernie étranglée sont classiques en chirurgie. Attendez : "patient avec tuméfaction inguinale irréductible douloureuse depuis X heures → que faites-vous ?" Réponse : urgence chirurgicale. Retenez que la crurale s'étrangle souvent.
4
⭐⭐ BRÛLURES — Classification par degrés
★★★★☆
Sujet nouveau introduit en session 2 → donc fort probable à l'examen de session 2. Concentrez-vous sur : 3ème degré = indolore (contre-intuitif mais très piégeur), phlyctènes = 2ème degré, règle des 9 pour l'étendue. Question classique : "brûlure avec zones indolores → quel degré ?" = 3ème degré.
5
⭐⭐ VOLVULUS & ASP — Images radiologiques
★★★☆☆
Le volvulus est régulièrement vu en exam car il a des signes radiologiques précis (bec d'oiseau) et un tableau clinique spécifique (vieillard + distension asymétrique). Les images ASP peuvent être montrées pour identification.
6
⭐⭐ SYNDROME TUMORAL — Localisations par quadrant
★★★☆☆
Moins de cas cliniques pratiqués mais les localisations par quadrant sont demandées en QCM. Retenez : FID = appendice + caecum + kyste ovaire D. FIG = sigmoïde + ovaire G. Masse pulsatile épigastre = anévrisme aorte. Contact lombaire = masse rénale.
7
📌 CONSEILS FINAUX STRATÉGIQUES
1. Maîtrisez les "pièges classiques" : 3ème degré brûlure = indolore (pas douloureux !), Occlusion ≠ contracture systématique, Fonctionnelle = gaz dans rectum (+).

2. Lisez les cas cliniques attentivement : Le Dr Lubin donne souvent 3-4 données cliniques dont 1-2 distracteurs. Cherchez le signe pathognomonique (selles gelée de groseille = invagination, asymétrie = volvulus).

3. Les "1ère cause" : Brides = 1ère cause occlusion mondiale. Iléo-colique = 1ère invagination fréquente.

4. Anatomie clef : Angle de Treitz, valvule de Bauhin, mésentère vs mésocôlon → attendez des QCM là-dessus.

5. Examen de choix : ASP pour occlusion générale / Échographie pour invagination / Lavement pour volvulus + invagination (diagnostique ET thérapeutique).
🎓 Message final : Ce cours est dense mais structuré. Les 7 thèmes s'articulent autour de 2 axes : urgences abdominales (occlusion, invagination, volvulus, hernies) et sémiologie générale (tumoral, brûlures). L'examen sera probablement 4 QCM/cas sur les urgences abdominales et 1-2 cas sur les brûlures/tumoral. Bon courage, tu as les bases, maintenant c'est la répétition qui fait la différence !
